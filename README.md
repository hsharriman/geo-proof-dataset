**Naming Convention**

Format: `textbookPrefix_section_problemTypeProblemNumber_solutionLabel.txt`

Problem types:
* `ex`: example
* `exer`: exercise
* `cio`: Check It Out

Example: `holt_s4-2_exer27_c1.txt` means:
* `holt`: Holt textbook
* `s4-2`: Section 4.2
* `exer27`: Exercise 27
* `c1`: Correct solution #1

For incorrect proofs, use `inc1`, `inc2`, etc. For example: `holt_s4-2_exer27_inc1.txt`
